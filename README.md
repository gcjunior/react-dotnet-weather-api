# Project Title

React web app that loads data from a .Net application that pulls weather details from openweathermap API.

## Description

It took me 6hrs to develop the whole application using AI as a tool.

## Getting Started

In order to have the application running you need to run the .net app and than the react app.

### Installing

.Net App:

```
dotnet run
```

React App:

```sh
npm install 

OR

yarn install
```

### Executing program

```
npm start --reset-cache
```

Access url: http://localhost:3000/
